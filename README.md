# moMoney MVP

moMoney is an embedded, optional acclimation layer designed to reduce friction at the highest-failure moment in retail investing: **first funding and first trade**.

This repository contains a lightweight MVP used to test a single behavioral hypothesis through a clean, falsifiable experiment.

## Deployment

**Live:** https://typeshi.vercel.app

**Test iframe embedding:** Open `test-embed.html` locally after deployment.
